## Descripción del Proyecto
Admitix AI es una solución tecnológica diseñada bajo el modelo de Software como Servicio o SaaS que actúa como un asistente de admisiones inteligente para la Secundaria Particular Lomas del Valle. La plataforma optimiza los procesos de captación y atención conversacional de la institución mediante la integración de un agente inteligente automatizado en la primera línea de respuesta comercial. 

El sistema extrae de forma automática las variables esenciales del usuario, tales como el nombre del tutor y el grado de secundaria de interés, transmitiendo esta información en tiempo real mediante peticiones seguras hacia un servidor externo para su posterior seguimiento administrativo.

## Tecnologías Principales Empleadas
* **Voiceflow**: Desarrollo del flujo del agente y control de variables locales.
* **OpenAI API**: Procesamiento de lenguaje natural mediante modelos GPT.
* **Webhook.site**: Endpoint y servidor de pruebas virtuales para la recepción segura de peticiones HTTP POST.
* **PlantUML**: Generación y renderizado del diagrama técnico de actividades.
